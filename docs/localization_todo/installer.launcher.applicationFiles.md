# installer.launcher.applicationFiles

- **English value**: `Applications and shortcuts`
- **Namespace**: `installer`
- **File/component**: `src/lib/tauri.ts`
- **UI role**: `label`
- **User flow**: The native file picker uses this label for its executable and shortcut file filter.
- **Tone**: `concise/neutral`
- **Placeholders**: `none`
- **Context/meaning**: Applications are Windows launchable files; shortcuts specifically means `.lnk` shell shortcuts.
- **Domain notes**: This labels a native file-type filter for `.exe`, `.com`, `.bat`, `.cmd`, and `.lnk` files.
