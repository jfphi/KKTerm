# installer.launcher.recentFolders

- **English value**: `Recent folders`
- **Namespace**: `installer`
- **File/component**: `src/modules/installer/InstallerToolDialog.tsx`
- **UI role**: `label`
- **User flow**: Heading above the remembered project-folder list in the mini launcher dialog for installed coding-agent CLIs (Claude Code, Codex, Kimi Code, Grok Build, OpenCode); clicking a folder opens a terminal there.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Context/meaning**: "Recent" = most recently opened by the user; "folders" = filesystem directories (project folders), not UI folders or mail folders.
- **Domain notes**: Folder paths themselves render untranslated.
