# webview.openLinkInNewTab

- **English value**: `Open link in new Tab`
- **Namespace**: `webview`
- **File/component**: `src/modules/workspace/connections/webview/WebViewWorkspace.tsx`
- **UI role**: `label`
- **User flow**: Shown in the URL Connection page context menu when the pointer is over an HTTP(S) link.
- **Tone**: `concise/neutral`
- **Placeholders**: `none`
- **Context/meaning**: Opens the targeted page link as another KKTerm Workspace Tab, not an external browser tab.
- **Domain notes**: `Tab` is the frontend Workspace container defined by KKTerm domain language.
